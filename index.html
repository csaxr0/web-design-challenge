Summary: Latitude vs. temperature, cloudiness, wind speed, and humidity

This project analyses how weather changes as you get closer to the equator. To accomplish this analysis, using data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After collecting the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we considered included: temperature, cloudiness, wind speed, and humidity. This page provides the source data and visualizations created during the analysis, as well as explanations and descriptions of any trends or correlations.
